Spring Boot / Spring Framework, Thymeleaf, Spring Data (MongoDB, MySQL), Microservicios, AWS
 *
 * Descripción:
 *   ClotFilms desea implementar una aplicación web para distribuir películas.
 *
 * Objetivos:
 *   1. Desarrollar aplicación web con Java y Spring Boot
 *   2. Autenticación con OAuth2 (https://oauth.net/2/)
 *   3. Persistencia de datos en MongoDB y MySQL
 *   4. Motor de plantillas Thymeleaf
 *   5. Paradigma de microservicios (Maven)
 *   6. Despliegue en AWS
 *
 * Requerimientos funcionales:
 *   - Autenticación OAuth2 para acceder al sistema
 *   - Usuarios pueden seleccionar películas; datos almacenados en MongoDB
 *   - CRUD completo para películas (crear, leer, actualizar, borrar)
 *   - Parte de administración para gestionar usuarios y sus películas consultadas
 *
 * Requisitos generales:
 *   - Análisis y diseño OO previo; diagrama de clases UML
 *   - Implementación con Spring Boot
 *   - Clases entidad y utilidades
 *   - Patrón MVC
 *   - Microservicios de datos y autenticación segura
 *   - Documentación JavaDoc en todas las clases
 *   - Diseño responsive
 *
 * Criterios de evaluación (puntuación total 10, mínimo 5):
 *   - Mockup, requisitos y diagrama UML: 5%
 *   - Autenticación OAuth2: 10%
 *   - Desarrollo Spring y funcionalidades (películas, iconos, administración): 25%
 *   - Spring MVC, Thymeleaf, atributos personalizados y página de errores: 25%
 *   - Persistencia y microservicios (MySQL Spring Data, MongoDB Spring Data): 25%
 *   - Despliegue (Docker, AWS): 10%
